# 通州智慧文旅 AI 大数据平台
## 项目介绍
通州区文旅数据专区展示平台，包含景区展示、客流监测、票根经济、数据可视化、AI 数据分析能力，适配电脑、Mac、手机访问。

## 技术栈
- HTML5：页面结构
- CSS3：样式与响应式布局
- JavaScript：交互逻辑
- Chart.js：客流趋势折线图
- 本地图片：真实景区图
- GitHub Pages：线上公网访问

## 项目文件结构
smart-travel/
├─ index.html          项目主页面
├─ css/style.css       样式文件（科技风格 + 手机适配）
├─ js/index.js         交互文件（导航、弹窗、图表、API）
├─ images/             本地真实景区图片
└─ README.md           项目说明文档

## 核心功能
1. 科技风 AI 大数据界面
2. 通州五大景区真实图片展示
3. 点击景区卡片弹出实时数据
4. 近 7 日全域客流趋势折线图
5. 导航点击高亮
6. 弹窗可关闭（点×、点空白、按 ESC）
7. 电脑 / Mac / 手机全部适配
8. 本地图片加载，断网也能看
9. 支持对接真实文旅 API

## 本地运行方法
1. VS Code 打开 smart-travel 文件夹
2. 点击右下角 Go Live
3. 浏览器自动打开：http://127.0.0.1:5500
